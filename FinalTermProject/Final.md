- README
	- **프로젝트명:** 청주대학교 캠퍼스 주차 안내 시스템
    
- **개발 목적:**  
    청주대학교를 방문하는 차량이 자신의 차량 유형과 방문 건물을 기준으로 최적의 주차장을 추천받고, 해당 주차장의 세부 정보를 확인할 수 있도록 돕는 콘솔 기반 프로그램을 제작한다.

- 프로젝트의 구조:
	- 프로젝트의 구조로는 Main.java (메인 실행 클래스 (입력 및 흐름 제어))
	- ParkingManager.java (주차장 추천 및 메인 로직)
	- ParkingLot.java(주차장 정보를 저장하는 데이터클래스)


- Main.java 설명
	-  전체 프로그램의 흐름을 제어
    
	- 사용자 입력에 따라 차량 유형, 충전 필요 여부, 건물 번호 등을 입력받음
    
	- 입력값 검증 (3회 실패 시 프로그램 종료)
    
	- 주차장 추천 결과 출력 및 세부 정보 


- ParkingManager.java
	- - 주차장 객체를 리스트에 저장
    
	- 건물 번호에 따른 추천 주차장 매핑 (Map<Integer, List<String>>)

		차량 유형에 맞는 주차장 필터링 및 추천 출력
    
		getLotById()로 주차장 ID에 따른 상세 객체 반환

   - ParkingLot.java
	   -각 주차장의 정보 보유: 전기차/트럭/일반차 자리 수, 충전기 여부, 지하 여부, 높이 제한, 인접 건물 등
    
	   -showDetails()로 주차장 정보 출력


순서도 시각화

	 - 차량 종류 입력 (전기차 / 승용차 / 트럭)
    
	- 전기차의 경우 충전 필요 여부 확인
    
	- 방문 건물 번호 입력
    
	- 주변 추천 주차장 목록 출력
    
	- 주차장 ID 입력 → 세부 정보 출력
    
	- 3회 이상 잘못 입력 시 오류 메시지와 함께 종료
